# BlinkIT Sales Analysis Dashboard (Power BI)

## Project Overview
This project is a **Power BI sales analytics dashboard** built using the BlinkIT grocery dataset. The dashboard analyzes **sales performance, product categories, outlet types, and customer preferences** to generate business insights.

The goal of this project is to demonstrate **data analysis, data modeling, and interactive dashboard development using Power BI**, which are key skills for data analyst roles.

---

## Objectives
- Analyze total sales and product performance.
- Identify top-performing product categories.
- Compare sales performance across outlet types.
- Understand the impact of outlet size and location on sales.
- Provide interactive insights using filters and visualizations.

---

## Tools & Technologies
- Power BI
- Power Query (Data Cleaning)
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Dashboard Design

## Dashboard Features

### 1. Total Sales by Fat Content
Shows the distribution of sales between:
- Low Fat products
- Regular products

This helps understand customer product preferences.

---

### 2. Sales by Item Type
Displays sales performance across product categories such as:
- Fruits and Vegetables
- Snack Foods
- Dairy
- Frozen Foods
- Household
- Baking Goods

This helps identify the most profitable product categories.

---

### 3. Outlet Establishment Trend
A line chart showing how sales vary based on the **year outlets were established**.

This helps understand store performance over time.

---

### 4. Sales by Outlet Size
Analyzes sales distribution across:
- Small outlets
- Medium outlets
- Large outlets

This helps determine which store size generates the most revenue.

---

### 5. Sales by Outlet Location
Compares sales across different city tiers:

- Tier 1
- Tier 2
- Tier 3

This helps analyze geographic sales performance.

---

### 6. Outlet Type Analysis
A table showing performance of different outlet types including:

- Grocery Store
- Supermarket Type 1
- Supermarket Type 2
- Supermarket Type 3

Metrics included:
- Total Sales
- Number of Items
- Average Sales
- Average Rating
- Item Visibility
